## Project Structure

```
kwamboxx-media-kit/
├── index.html          # Main HTML — structure only, no inline styles or scripts
├── css/
│   └── styles.css      # All styling — typography, layout, animations, colours
├── js/
│   └── main.js         # All JavaScript — starfield, comets, card flips, scroll reveals
├── assets/
│   └── images/         # Portrait and lifestyle photos (currently embedded as base64)
├── README.md
└── LICENSE
```



## Tech Stack

- Vanilla HTML5 / CSS3 / JavaScript (no frameworks, no dependencies)
- Google Fonts: Cormorant Garamond, Outfit, Space Mono
- No build step required — open `index.html` directly in any browser
